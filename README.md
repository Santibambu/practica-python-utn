# Prácticas de Python – Programación I (UTN FRA)

Repositorio que reúne ejercicios de práctica desarrollados durante la cursada de la materia **Programación I** de la **UTN Facultad Regional Avellaneda**.

Los ejercicios están organizados por temática e incluyen prácticas generales, ejercicios integradores y modelos de parcial.

## Contenidos trabajados
- Condicionales y estructuras de control
- Ciclos (`for`, `while`)
- Funciones y modularización
- Listas, listas anidadas y diccionarios
- Recursividad
- Algoritmos de ordenamiento
- Manejo de archivos (CSV, JSON)
- Paradigma funcional
- Resolución de ejercicios integradores y parciales

## Estructura del repositorio
Cada carpeta corresponde a un tema específico y contiene uno o más ejercicios relacionados, organizados de forma modular cuando corresponde.

Las prácticas de parcial y parciales se encuentran agrupadas en carpetas específicas para facilitar su identificación.
